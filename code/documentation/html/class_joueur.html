Member Enumeration Documentation

enum Joueur::states
private

ensemble des différents états du Joueur (utilisé surtout pour les sprites)

Author
Quentin CHEYNET
Enumerator:
STANDING 

le personnage est debout, sur le sol, et ne fait rien

RUNNING_R 

le personnage court vers la droite

RUNNING_L 

le personnage court vers la gauche

FALLING 

le personnage est en train de tomber

LANDING 

le personnage atterit sur le sol

GETTING_UP 

le personnage se relève après avoir atteri sur le sol

JUMPING 

le personnage est dans les air après un saut, avant d'avoir atteint sa hauteur maximal

DROPING 

le personnage lache une bombe

DYING 

le personnage est en train de mourrir

DEAD 

le personnage est mort

WINNING 

le personnage a gagné et s'en réjouit
